## How the calculators work (overview)

### Pip Value
The pip value calculator estimates how much a single pip movement is worth based on:
- Trading pair
- Lot size
- Account currency

This helps traders understand potential profit or loss per pip.

### Lot Size & Risk
Lot size and risk calculators help estimate position size based on:
- Account balance
- Risk percentage
- Stop loss distance

These tools are commonly used in basic risk management.

### Margin
The margin calculator estimates required margin using:
- Lot size
- Leverage
- Instrument price

This helps traders avoid over-leveraging.
